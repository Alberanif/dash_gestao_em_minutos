Design — Fluxo de dados (atualizado)

O cron lê apenas contas já cadastradas. Não cria, não descobre contas novas.

Fluxo de coleta — Cron Job
⏱ Cron dispara
POST /api/cron/collect
→
Lê contas já cadastradas
SELECT * FROM accounts WHERE is_active = true
⚠ Não descobre nem cria contas novas
Para conta YouTube:
collectYouTube(account.credentials)
→ usa api_key + channel_id das credentials
Para conta Instagram:
collectInstagram(account.credentials)
→ usa access_token + user_id das credentials
Salva snapshot no Supabase
INSERT com account_id + collected_at
→
Registra em
cron_logs
Responsabilidades separadas
✅ Cron faz
  • Lê contas ativas do banco
  • Coleta dados de cada conta via API
  • Persiste snapshots com account_id
  • Registra log de execução
❌ Cron NÃO faz
  • Descobrir novas contas
  • Criar ou editar contas
  • Validar tokens
